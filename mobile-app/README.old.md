# Mobile App

React app for scanning QR codes and displaying vouchers in real-time.

## Features
- Scans QR code using device camera
- Sends sessionId to backend
- Receives and displays voucher list
- Responsive, clean UI

## Setup
1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the app:
   ```sh
   npm start
   ```
